原本只计划讲connect和express，在看源码的过程中手痒了，写了个connect-header，还发布了；所以干脆把module的开发一起拉进来。

module全称Node Packaged Module，很多语言也有类似的概念，如：python、perl

connect-header源码在github，也可以直接安装：

npm install connect-header
